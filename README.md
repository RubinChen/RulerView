RulerView
==========
喝水不忘挖井人，改造自https://github.com/pruas/Ruler！
自定义尺子：
--------
  1，支持水平竖直;<br>
  2，支持设置尺子刻度，数值大小与颜色;<br>
  3，支持刻度间距;<br>
  4，支持是否显示特殊刻度的特殊颜色与尺寸（整5格，整10格）;<br>
